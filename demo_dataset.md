# PersonaLabs Curated Demo Dataset

Use high signal-richness videos for repeatable demos. Weak metadata creates weak explainability, so random browsing is best reserved for uncertainty demos.

PersonaLabs is not truth detection, misinformation detection, political moderation, or censorship. PersonaLabs is explainable cognitive observability, attentional friction modeling, intentional alignment exploration, and persona-weighted media analysis.
